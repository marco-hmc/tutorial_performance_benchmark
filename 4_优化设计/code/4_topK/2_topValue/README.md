题目：在 100w（1,000,000）个数中找出最大的 100 个数。

示例中为便于演示使用较小数据量并输出 Top-10（可在源码中调整 K）。

三种方案：

- 方案 1（局部淘汰法）：先选前 K 个排序为有序序列 L，依次扫描剩余元素，若当前元素大于 L 中最小元素，则替换并插入到正确位置（插入排序思想）。
- 方案 2（基于快速选择 / QuickSelect）：使用快速选择的思想，递归分区只关注比 pivot 大的那一半，直至找到前 K 大的集合，然后排序取 Top-K。
- 方案 3（使用最小堆）：维护大小为 K 的最小堆，遍历数据时用当前元素替换堆顶（若更大），最后堆内即为 Top-K。

目录内容：

- `top100_100w.cpp`：三种方案实现，带中文注释。
- `sample_numbers.txt`：样例输入（小规模）。
- `run_test.sh`：编译并在样例上运行，演示三种方案（当前 K=10）。
