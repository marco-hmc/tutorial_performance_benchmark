题目：在一个大整数集合中找出只出现一次的整数（2-bit 位图方法示例）

说明：

- 输入：一个包含无符号 32-bit 整数的文本文件，每行一个数字。数据量可能非常大，内存受限。
- 目标：列出在输入中只出现一次的所有整数。
- 方法：使用 2-bit 位图（00=未出现，01=出现一次，10=出现多次）来标记每个整数的出现次数。对于 32-bit 值空间，该位图占用 2 \* 2^32 bits ≈ 1 GiB 内存（示例中将限制数值范围以便在测试机上运行）。

本目录包含：

- `unique_once.cpp`：C++ 实现（以文件流逐行读取，构造 2-bit 位图，并输出只出现一次的整数）。
- `test_input.txt`：一个小型测试输入文件。
- `run_test.sh`：编译并运行测试脚本，输出结果到 stdout。

注意：为便于本地测试，程序支持通过 `-m` 参数限制最大数值范围（例如 `-m 1000` 表示仅考虑 [0,999] 的值并分配相应位图）。
