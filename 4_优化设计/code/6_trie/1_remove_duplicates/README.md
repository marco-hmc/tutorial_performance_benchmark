# 题目：去除 1000 万字符串中的重复项（示例实现）

问题描述：给定大量字符串（例如 1000 万条），要求去重并保留不重复的字符串集合。内存可能有限，常见方案包括使用 `trie` 或 `hash_map`。

本示例提供两种实现（用于小规模演示）：

- `hash_map` 方法：直接用 `unordered_set` 判断是否已存在并输出不重复字符串（适用于内存足够的情况）。
- `trie` 方法：使用简单的 Trie 结构逐词插入并判断是否首次出现（节省哈希开销、适合前缀密集场景）。

运行：

```bash
cd 5_数据设计/code/9_remove_duplicates
./run_test.sh
```
