题目：8G 的 int 数据，内存只有 2G，如何排序？（外部排序）

本示例演示外部排序的基本流程（分块->内部排序->多路归并），用于学习和验证思路。

流程：

1. 将大文件按块划分（每块大小应能放入内存），写入临时子文件。
2. 对每个子文件在内存中排序（内部排序），输出有序子串文件。
3. 使用多路归并（k-way merge）合并所有有序子串，得到最终有序文件。

本示例为演示用途使用小文件和小内存阈值，可以在 `external_sort.cpp` 中调整 `CHUNK_SIZE`。

包含文件：

- `external_sort.cpp`：实现演示外部排序的程序（带中文注释）。
- `sample_large.txt`：小规模样例输入。
- `run_test.sh`：编译并运行示例。
